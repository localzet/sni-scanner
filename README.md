# 🛸 SNI SCANNER

**Advanced Network Diagnostic Tool for Android**

SNI Scanner — это мощное приложение для Android, предназначенное для глубокой сетевой диагностики, проверки SNI (Server Name Indication), TLS-сертификатов и доступности хостов. Идеально подходит для сетевых инженеров, системных администраторов и любых пользователей, которым необходимо проверить доступность доменов через конкретный IP-адрес.

---

## ✨ Ключевые возможности

### 🔍 3 Режима работы
1. **Свой SNI (Single)** — Проверка одного SNI-хоста через указанный IP, с максимально подробным отчётом по всем полям (Geo, TCP, TLS, HTTP, DNS, сертификаты).
2. **Whitelist** — Автоматическая загрузка актуального списка SNI (локальный кэш + тихое обновление при старте), пакетная проверка всех доменов с подробным логом в терминале.
3. **Massive** — Ручной ввод списка SNI или загрузка `.txt` файла одной кнопкой; массовое тестирование со сводным отчётом.

### ⚡ Fast mode (2–10 потоков)
Для режимов **Whitelist** и **Massive** можно включить параллельную проверку:
- Чекбокс **⚡ Fast (может снизить точность)** в панели управления списком.
- Слайдер одновременных потоков от **2 до 10**.
- Реализовано на `kotlinx.coroutines.sync.Semaphore` — соблюдает паузу/отмену, выводит результаты по мере готовности.
- На 5–10× быстрее последовательной проверки, ценой небольшой погрешности за счёт общей сетевой нагрузки.

### 📥 Whitelist-кэш и первый запуск
- При первом запуске показывается **невынимаемый диалог** с предложением скачать whitelist из GitHub.
- Файл сохраняется в `filesDir` приложения — проверки работают **мгновенно и офлайн**.
- На каждом старте приложение тихо проверяет обновления; если сети нет — используется кэш.
- Метаданные (последняя дата обновления, число записей) хранятся в SharedPreferences.

### 🌐 Работа в фоне
- Проверки теперь переживают сворачивание приложения: запускается **foreground-сервис** (`FOREGROUND_SERVICE_TYPE_DATA_SYNC`).
- Запрашиваются разрешения `POST_NOTIFICATIONS` (Android 13+) и отключение оптимизации батареи.
- Уведомление живое и обновляется каждую секунду:
  - Прогресс `X / Y · %` в subText,
  - Текущий SNI — прошло/осталось времени в основном тексте,
  - Полная сводка в развёрнутом виде (`BigTextStyle`): текущий SNI, прогресс, прошло, осталось, режим (`⚡×N` или последовательный), `⏸ PAUSED`.
- Тап по уведомлению возвращает в приложение.

### 📤 Экспорт CSV
- Кнопка **Export CSV** во всех трёх режимах.
- Сохранение через Storage Access Framework в любую папку.
- Экспортируются **все поля** `CheckResult` + `IpGeoInfo`: geo, DNS (системный + DoH Google/Cloudflare), все TCP-порты, TLS-версии, сертификат (SAN-лист, issuer, daysToExpiry), HTTP-заголовки, ICMP, вердикт и ошибки.

### 🔒 Анализ TLS и Сертификатов
- Поддержка **TLS 1.0 / 1.1 / 1.2 / 1.3** (с детекцией поддержки каждой версии отдельно).
- Проверка **HTTP/2 (ALPN)**.
- **SNI Match** — верификация соответствия запрошенного SNI и данных в сертификате.
- Детали сертификата: subject, issuer, notBefore/notAfter, дни до истечения, self-signed флаг, полный SAN-list.
- **DPI-детектор**: сравнение TLS с SNI и без SNI для обнаружения фильтрации.

### 🔌 TCP-диагностика
- Порты: 443, 80, 8443, 8080, 22 (SSH), 25 (SMTP), 465 (SMTPS), 53 (DNS/TCP).
- Время установки соединения, тип блокировки (`RST` / `TIMEOUT`).
- Измерение **RTT (ping)** через TCP-connect.

### 🔍 DNS-анализ
- Системный резолвер + **DNS-over-HTTPS** через Google (8.8.8.8) и Cloudflare (1.1.1.1).
- Проверка согласованности ответов и **детекция DNS-отравления**.
- Reverse DNS (PTR).

### 🌍 Геолокация и IP Info
- Fallback-цепочка: `ipinfo.io → ipapi.co → ipwhois.app → freeipapi.com → ip-api.com`.
- Кэш результатов в памяти.
- Поля: organization, ASN, city, region, country/code, postal code, timezone, hostname, anycast, координаты.

### 🎛️ UI и Контроль
- **Material 3 Design** с поддержкой динамических тем.
- **Material-диалоги** с иконками (whitelist download, permission, battery, language).
- **Тёмная / Светлая тема** — переключение одной кнопкой.
- **Многоязычность** через Google ML Kit — перевод интерфейса на лету.
- **Пауза / Резюме / Отмена** с корректной остановкой фоновой задачи.
- **Фильтр `≤100мс`** для скрытия медленных хостов.
- **Интерактивный терминал** с подробным логом (префикс `⚡` для fast-mode, стрелки `→ / ←` для whitelist).
- IP по умолчанию — **1.1.1.1**.

## 🛠 Технологический стек
- **Язык:** Kotlin
- **Асинхронность:** Coroutines, `kotlinx.coroutines.sync.Semaphore`, AtomicInteger
- **UI:** View Binding, Material 3, Motion/Transitions
- **ML:** Google ML Kit (On-device Translation)
- **Network:** Raw Sockets, `HttpsURLConnection`, DNS-over-HTTPS
- **Storage:** Internal files (`filesDir`), SharedPreferences, Storage Access Framework (CSV)
- **Background:** Foreground Service с `dataSync` типом, `NotificationCompat.BigTextStyle`

---

## 📦 Разрешения
- `INTERNET`, `ACCESS_NETWORK_STATE` — сетевые проверки.
- `POST_NOTIFICATIONS` — прогресс в фоне (запрашивается на Android 13+).
- `FOREGROUND_SERVICE`, `FOREGROUND_SERVICE_DATA_SYNC` — живое уведомление.
- `WAKE_LOCK` — предотвращение засыпания во время длительных проверок.
- `REQUEST_IGNORE_BATTERY_OPTIMIZATIONS` — предложение отключить оптимизацию батареи (опционально, по согласию пользователя).

## 🚀 Как пользоваться

1. **При первом запуске** подтвердите скачивание whitelist (нужно один раз, дальше работает офлайн).
2. **Введите целевой IP** (по умолчанию `1.1.1.1`).
3. **Выберите режим**: `Single` / `Whitelist` / `Massive`.
4. Для массовых режимов опционально:
   - Отметьте **⚡ Fast** и выберите число потоков (2–10).
   - В Massive — нажмите иконку загрузки на поле списка, чтобы загрузить `.txt` (по одному домену на строку, `#` — комментарий).
5. **Запустите проверку** (кнопка `Check`). Свернуть приложение можно — прогресс будет в уведомлении.
6. **Экспорт:** кнопка `Export CSV` сохранит полный отчёт со всеми полями.

## 📁 Архитектура

```

├── MainActivity.kt            — UI, режимы, диалоги, fast mode, тикер уведомлений

├── CsvExporter.kt             — CSV-сериализация результатов

```
